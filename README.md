# LeetCode Solutions — Java

Repositório pessoal com soluções para problemas do [LeetCode](https://leetcode.com/) implementadas em Java.

---

## Estrutura do projeto

```
leet-code/
├── src/
│   ├── twoSum/
│   ├── validParentheses/
│   ├── longestCommonPrefix/
│   ├── mergeTwoSortedLists/
│   ├── palindromeNumber/
│   ├── removeDuplicatesfromSortedArray/
│   ├── removeElement/
│   └── romanToInteger/
└── LeetCode.iml
```

Cada pasta corresponde a um problema e contém a classe Java com a solução e casos de teste no método `main`.

## Tecnologias

- **Java 8+**
- **IntelliJ IDEA**

---

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/leet-code.git
   ```

2. Abra o projeto no IntelliJ IDEA (o arquivo `LeetCode.iml` já está configurado).

3. Navegue até a classe desejada dentro de `src/` e execute o método `main`.
